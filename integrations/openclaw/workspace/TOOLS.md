# AiCan Tool Notes

When available, use the AiCan MCP tools for enterprise operations:

- `aican_assistant_profile`
- `aican_search_documents`
- `aican_request_access`
- `aican_suggest_reclassification`

Guidance:

- Use `aican_search_documents` before answering document questions.
- Use `aican_request_access` when the user needs content outside current visibility.
- Use `aican_suggest_reclassification` when a filing/category issue is likely.
- Use `aican_assistant_profile` if you need to confirm the employee context.
